Frequently Asked Questions

What if a user isn't signed into Google?

We automatically optimize which login options to show each visitor, based on which networks they're already signed into. We always have a manual e-mail address text field also available as a default.

How do I send the signups to my e-mail marketing tool like Mailchimp, ActiveCampaign, etc?

We have built in integrations with all the major e-mail marketing tools, so all you need is your credentials. For other integrations, we work with Zapier to allow you to setup automated workflows with over 1,000 other apps.

How is this different from social sign-in buttons?

Unlike the normal sign-in with Facebook buttons you may have seen around the web, our login options all leverage 1 or 2-click sign-in options that only became available in 2020. Normal login options typically navigate the browser away from the page you're on, which isn't ideal for e-mail subscription flows.

How do I install the website popup to get subscribers?

We have plugins available for Shopify, Wordpress, and other platforms that install in under a minute - no IT required.
